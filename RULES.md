# 规则书：我的世界卡牌游戏

本规则书用于指导完整的游戏流程与卡牌交互。所有道具、怪物与方块设计均参考《我的世界》现有元素。

## 1. 组件

- **主牌库**：资源、方块、工具、武器、护甲、事件、建筑、附魔。
- **生物群系牌库**：森林、沙漠、雪地、下界、末地。
- **天气牌库**：晴天、雨天、雷暴、风雪、烈日。
- **Boss 牌**：末影龙、凋灵。
- **标记物**：生命、护甲、饥饿、能量、经验、耐久、建筑稳定度。

## 2. 玩家数与目标

- **玩家数**：1-4（可合作对抗 Boss）。
- **胜利条件**：
  - 击败末影龙或凋灵；
  - 或建造并激活“末地传送门”。
- **失败条件**：
  - 生命值归零；
  - 或连续三回合饥饿为 0。

## 3. 起始设置

- 生命 20、护甲 0、饥饿 6、能量 3、经验 0。
- 起手抽 5 张卡。
- 随机翻开 1 张生物群系与 1 张天气。

## 4. 回合流程

1. **环境阶段**
   - 结算生物群系与天气效果。
   - 若雷暴：掷骰 1-3 触发“闪电”事件（随机造成 1-2 点伤害或点燃建筑）。
2. **准备阶段**
   - 抽 1 张牌。
   - 能量恢复至基础值（默认 3，建筑可提升）。
3. **行动阶段**
   - 打出卡牌（消耗能量）。
   - 合成装备或建筑。
   - 与怪物战斗或使用事件卡。
4. **结束阶段**
   - 结算持续效果与建筑维护。
   - 若饥饿为 0，受到 2 点真实伤害。

## 5. 卡牌类型说明

- **资源牌**：提供合成材料点（木材、石头、铁锭、金锭、钻石）。
- **方块牌**：用于建造建筑或提升护甲值。
- **工具牌**：提高采集与抽牌效率。
- **武器牌**：提升攻击力，对怪物造成伤害。
- **护甲牌**：提高防御力并吸收伤害，耐久归零时损坏。
- **怪物牌**：若未击败，回合结束造成伤害。
- **事件牌**：即时效果，例如“苦力怕爆炸”。
- **建筑牌**：提供持续增益或合成折扣。
- **附魔牌**：为武器与护甲提供临时增强。

## 6. 合成系统

合成需消耗资源点，常见配方：

- **木板 + 木棒 = 木剑**
- **圆石 + 木棒 = 石剑**
- **铁锭 + 木棒 = 铁剑**
- **钻石 + 木棒 = 钻石剑**
- **铁锭 + 皮革 = 铁胸甲**
- **玻璃 + 铁锭 = 望远镜（抽牌 +1）**
- **黑曜石 x4 + 火焰点 = 下界传送门**

## 7. 生物群系与天气

| 生物群系 | 效果 |
| --- | --- |
| 森林 | 木材类卡牌费用 -1 |
| 沙漠 | 食物类卡牌效果 -1，怪物攻击 +1 |
| 雪地 | 第一张工具卡免费 |
| 下界 | 火焰、烈焰棒、岩浆相关卡牌效果 +1 |
| 末地 | 抽牌阶段额外抽 1 张，但怪物攻击 +1 |

| 天气 | 效果 |
| --- | --- |
| 晴天 | 无 |
| 雨天 | 水相关卡牌效果 +1 |
| 雷暴 | 结束阶段有概率触发闪电 |
| 风雪 | 食物回复效果 -1 |
| 烈日 | 资源采集 -1，但怪物攻击 -1 |

## 8. Boss 战

- **末影龙**：攻击 7，护甲 5，拥有“飞行”——每回合免疫 1 次攻击。
- **凋灵**：攻击 6，护甲 6，回合结束造成 1 点额外真实伤害。

## 9. 经验与升级

- 击败怪物获得经验。
- 每累计 5 点经验，提升 1 级：能量上限 +1 或护甲 +1。

## 10. 合作模式（可选）

- 玩家共享 Boss 生命，但各自维持生命与资源。
- 支援行动可将 1 张手牌交给队友。

## 11. 高阶玩法策略（提升趣味性）

### 11.1 人物升级与职业分支

- **升级方式**：每获得 5 点经验可升级一次，升级时二选一：
  - **战士**：攻击 +1，武器耐久 +1（更偏向怪物清场）。
  - **工匠**：合成所需资源 -1（最低为 1），建筑耐久 +1（更偏向建筑与经济）。
  - **探险家**：抽牌 +1 或行动阶段第一张事件卡免费（更偏向牌效与控制）。
- **升级限制**：每个角色最多 3 次升级，防止数值失衡。

### 11.2 碎片合成系统

- **掉落规则**：击败怪物或完成事件可获得“碎片”。
- **合成阈值**：收集 3 个同类型碎片可合成对应稀有卡（如末影珍珠、附魔书）。
- **策略价值**：碎片鼓励玩家优先选择特定事件或怪物，形成风险与奖励的平衡。

### 11.3 动态事件链

- **事件链机制**：触发特定事件后，在未来 1-2 回合内解锁后续事件牌。
  - 例：触发“矿洞塌方” → 下一回合可能出现“救援矿工”或“遗失宝箱”。
- **策略价值**：玩家可以“押注未来收益”，在短期亏损中换取强力奖励。

### 11.4 建筑升级与防线

- **建筑升级**：建筑牌可叠加升级（如“简易基地 → 石制堡垒 → 附魔防线”）。
- **防线机制**：建筑拥有“防线值”，能抵消怪物溢出伤害。
- **策略价值**：鼓励防御型玩家在关键回合建立“保命墙”。

### 11.5 稀有资源的风险开采

- **开采模式**：玩家可选择“安全开采”（稳定+1资源）或“深层开采”（掷骰，成功获得稀有资源，失败触发怪物）。
- **策略价值**：风险开采为后期强力装备铺路，促进紧张感。

### 11.6 Boss 多阶段与弱点机制

- **多阶段**：Boss 生命降至一半时进入第二阶段，攻击 +1，新增特性。
- **弱点触发**：某些卡牌（如“弓箭”或“附魔穿透”）可触发弱点，使 Boss 受到额外伤害。
- **策略价值**：玩家需要在前期准备特定卡组与道具应对。

### 11.7 牌库污染与净化

- **污染机制**：使用“诅咒”或“凋灵之息”等卡牌会将负面卡洗入牌库。
- **净化机制**：通过“神庙祈祷”或“附魔台”移除负面卡。
- **策略价值**：卡组构筑更具长期规划与代价管理。

## 12. 策略执行流程（示例回合）

以下示例展示如何在一局游戏中贯彻上述策略，强调“先铺资源与建筑，再找机会爆发”的节奏：

1. **前期（1-3 回合）**
   - 优先打出资源与工具卡，确保手牌与资源稳定。
   - 选择“安全开采”来保证基础合成所需。
   - 若出现碎片事件，尽量收集同类型碎片，为中期强力合成做铺垫。

2. **中期（4-6 回合）**
   - 升级到职业分支，推荐根据手牌选择“战士/工匠/探险家”。
   - 通过建筑升级构建防线，减少怪物牌的持续压力。
   - 如果抽到事件链的前置事件，优先触发以博取高收益。

3. **后期（7 回合以后）**
   - 使用碎片合成的稀有卡进行爆发（例如附魔书或末影珍珠）。
   - 进入 Boss 多阶段时，提前准备“弱点触发”卡。
   - 保持牌库净化，避免关键回合抽到负面牌。

此流程为通用策略示例，可根据生物群系与天气进行微调。

## 13. 动态合成策略与装备合成概率

### 13.1 动态合成策略

为避免玩法固化，合成策略可根据战局动态调整：

- **动态成本**：连续 2 回合未合成装备时，下一次合成成本 -1（最低为 1）。
- **资源波动**：若当前生物群系资源短缺（如沙漠木材少），可触发“替代配方”，用相近资源替代 1 个材料点。
- **事件影响**：触发“匠人来访”事件可使本回合任意合成额外获得 +1 耐久或 +1 攻击。

### 13.2 装备合成几率

合成装备可引入“成功率”和“品质”机制，使策略更具风险收益：

- **基础成功率**：
  - 普通装备：90%
  - 稀有装备：70%
  - 史诗装备：50%
- **失败补偿**：合成失败返还 50% 资源点或转化为“碎片”。
- **品质结果**：成功后掷骰决定品质（普通/精良/史诗），品质越高攻击或耐久加成越高。

### 13.3 影响合成几率的因素

- **建筑加成**：拥有“工作台”或“铁砧”时成功率 +10%。
- **职业加成**：工匠分支合成成功率 +15%。
- **附魔加成**：消耗“幸运”附魔可额外 +10%。

这些机制鼓励玩家在资源、建筑与职业之间做出权衡，从而形成更丰富的策略选择。

## 14. 合成特效与出牌卡片特效

为提升操作手感与反馈，增加以下特效规则（可在 UI 层实现）：

### 14.1 合成特效

- **合成成功**：出现“粒子爆闪”与“合成环”特效，附带音效“叮”。
- **品质提升**：精良/史诗品质会附加“星光飞散”或“彩虹火花”特效。
- **合成失败**：触发“碎裂”与“烟雾”特效，并将失败补偿以碎片形式弹出。
- **建筑加成合成**：若工作台或铁砧提供成功率加成，合成时显示“工匠印记”特效。

### 14.2 出牌卡片特效

- **资源/方块牌**：卡牌落地时出现“落砖”或“木屑飞散”特效。
- **武器牌**：打出时播放“挥砍轨迹”，若击杀怪物追加“爆裂击倒”特效。
- **护甲牌**：打出时触发“护盾展开”与“护甲闪光”特效。
- **怪物牌**：入场带“阴影扩散”或“低吼”特效，压迫感增强。
- **事件牌**：使用时出现“全屏提示闪烁”与对应主题特效（如爆炸、雷电）。
- **附魔牌**：打出时增加“符文环绕”特效，并短暂高亮被附魔装备。

这些特效可根据设备性能进行简化或关闭，确保低端设备也能流畅运行。
